================================================================================
                         BUILD & DEPLOYMENT
================================================================================

To use this project:

1. Place in your development directory:
   $ cp -r nabda_flutter ~/flutter_projects/

2. Install dependencies:
   $ cd nabda_flutter
   $ flutter pub get

3. Update Firebase:
   Edit lib/firebase_options.dart with your credentials

4. Run on emulator:
   $ flutter run

5. Build for release:
   $ flutter build apk        # Android
   $ flutter build ios        # iOS
   $ flutter build web        # Web
   $ flutter build macos      # macOS
   $ flutter build windows    # Windows
